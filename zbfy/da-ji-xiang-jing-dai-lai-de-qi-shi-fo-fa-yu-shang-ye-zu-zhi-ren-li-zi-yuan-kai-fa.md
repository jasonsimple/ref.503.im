# 《大吉祥经》带来的启示——佛法与商业组织人力资源开发

Buddhist Perspective for Organization Human Resource Development:

Reference to the Maha Mangala Sutta

作者：潘达玛斯瑞 M .KD

Padmasiri M.KD

作者介绍：人力资源管理系，商业和管理研究院，凯拉尼亚大学, 斯里兰卡

**摘要：**

在当今的商业世界中，人力资源开发（Human Resource Development, HRD）对于提高人力资源技能、正确态度和知识发挥着重要作用。HRD可以定义为一个专门组织为其成员设计一套系统的、有计划的活动，为其提供学习必要技能的机会，以满足当前和未来的工作需求。HRD的范围主要包括职业发展、个人发展、绩效管理和组织发展四个方面。本研究的目的是研究如何将《大吉祥经》的开示应用于个人发展（Individual Development, ID）的支分——技能、知识和行为的提升，以达到组织化HRD的目标。既往研究未充分解释与HRD学科相关的宗教观点，而本研究通过聚焦最著名的经论，探索其在HRD方面的意义来填补这一空白。本研究基于佛教三藏（相当于三“篮子”教义：经藏、律藏和论藏）和其他佛教典籍。《大吉祥经》宣说了三十八条有关个人发展的开示，本研究通过确定在组织层面上如何将这些开示分类纳入个人发展的支分，从而在佛教和个人发展之间架起了桥梁。如果在商业组织中对所选经中的这些开示进行创新式应用，将会为组织的成功带来重要作用。

**关键词：**HRD，《大吉祥经》，个人发展，技能，知识，行为

**一、背景介绍**

人力资源是竞争优势的源泉，人力资源不仅仅是员工履行自己的工作职责，而且要高效、有效地完成。要达到预期的水平，员工必须具备所需的知识和技能。HRD领域包含了组织授权给人力资源的几个方面。当前的HRD模式将员工视为价值创造者以及资产\[1\]。HRD是帮助员工发展个人能力、组织能力、以及知识的组织部门 \[2\]。HRD包括员工培训和发展、职业发展、绩效管理和个人发展等机会，并以此实现企业组织的发展。

HRD所有这些方方面面的核心是培养最优秀的劳动力，并最终实现组织的愿景。HRD是由组织设计的一整套有计划和系统的活动，为其成员提供学习必要技能的机会，以满足目前和未来的工作需求。HRD通过有组织性的开发、培训以及发展，来提升员工的专业技能\[3\]。HRD的目标是提高员工的绩效，其范围涵盖了四个主要方面：个人发展、职业管理、组织发展和绩效管理\[4\]。

世界上存在不同的宗教，其中最具影响力的有：基督教、佛教、天主教、印度教和伊斯兰教。在这些主要的宗教中，本研究仅限于由乔达摩所创建的佛教的观点，乔达摩也被称为乔达摩·悉达多、释迦牟尼、或直接简称为佛。佛是佛教的主要人物，他从不同方面所宣讲的教言散布于佛教不同宗派中 \[5\]。研究人员所选的《大吉祥经》，像《慈经》、《虎住者经》等经典一样论述了个人发展的内容，因此本研究选择佛所说的这部《吉祥经》作为研究对象。

**研究的意义**

斯瑞鹏\[6\]的研究清晰地解释了佛教如何有助于理解HRD，然而目前有关佛教观点的学术知识和人力资源实践较为少见。而本研究通过将佛教与HRD联系起来，填补了这方面的空白。此外也要说明，本研究的这种尝试并不是要去宣扬传播某种宗教，而是要在组织层面上探索收集佛教在HRD方面的实用性。因此，本研究的主要目的是探索如何利用佛教知识为HRD中的个人发展建立一套模式。

二、《大吉祥经》

根据佛教三藏 \[7\]，佛陀作了一场被命名为《大吉祥经》的开示，阐述了三十八种最吉祥的事情，其内容相当于详细介绍了一个人如何行持才能获得人生成就。除此之外，这三十八种吉祥之事在本质上是符合伦理和人性的，为我们的人生旅程提供了逐步成长的修行次第。研究人员首次尝试在索尼 \[8\]对此经的注释——《生命中最吉祥的事情》的英语译本基础上，详细阐述三十八种吉祥的含义。

**三、个人发展和《大吉祥经》**

通过研究，确定了《大吉祥经》中特别适合于个人发展的开示，而个人发展是我们在HRD范畴中选定的研究对象。根据艾格勒的说法 \[8\]，个人发展指的是学习掌握新知识、新技能、和/或改善行为，从而提高员工在当前工作（或培训）中的相关能力。本研究聚焦在组织背景下的个人发展上。

正如艾格勒所证明的 \[8\]： 个人发展的主要组成部分是开发新的知识、技能和行为改进。研究人员首先确定了个人发展的这三个要素，然后确定了与每个要素相关的开示。相应地，将“不与愚者交往、与智者交往、正确的指导思想”，归入个人发展中开发员工的技能和知识这个范畴。与此同时，包括以上三条开示在内，把“尊敬值得尊敬的人、优美和善的语言、布施、戒酒、谦卑、知足、感恩、安忍和接受建议”等九条开示，归入到个人发展中改善行为这一范畴，见下图。总之，研究人员从经典前三十二条开示中，选定了十二条开示归入个人发展的不同要素，具体见下图。

图1. 《大吉祥经》所论述的个人发展

1、不与愚者交往：根据图1发现，这条开示影响发展新知识、技能和改善行为。当一个人不与愚蠢的人交往时，他的知识、技能和行为就会提升。根据索尼的说法\[9\]，这一原则强调的是，远离而不是纠缠于那些带着愚者特征、在成长的岁月中早已丧失了童真、浑身充满失败和缺点的人。“这表明，当一个人与具有上述特征的人一起时，在组织层面上会有不同程度的劣势，例如他们可能会被引导参与不被社会接受的行为。”由此可以得出结论，在个人发展中，知识、技能和行为的发展都与此有关。因此，员工应与没有这些消极表现的人交往，以克服负面影响。

2、与智者交往：与上述开示相反的是与智者交往。与智慧同行的步骤：聆听良好的建议、理性的信心、高尚的思想、清晰的思考、自我控制、良好的行为、克服阻碍、获得智慧以及随之而来的解脱。只要做好自我控制，充满慈心和悲心，自然远离罪恶的侵害。进一步，与智者交往，可以使人更明智，由此发展知识、技能和行为；故员工应与智者交往。

3、正确的指导思想：人的行为可以体现在身体、言语或意识三方面，但归根到底都由心指导。如果我们的心里没有愿力，就不会有结果。根据佛陀的教言，重点在于“要自立自强”：一个人应该通过自己的努力来引导自己达到理想的目标。这其实是在鼓励人们要自信、不要依赖。他说，许多人在错误的道路上过着生活，他们在身体（行为）、言语和思想负面方面的邪行正是由于缺乏正确的指导思想造成的。按这个逻辑，当人的指导思想正确时，他就可以获得技能、知识和改善行为，而这也正是个人发展的主要要求。

4、尊敬那些值得尊敬的人：从商业角度，可以把尊敬理解为互相尊重。从商业角度来看，这一开示是维持良好关系的关键。对个人发展所要求的改善行为而言，应当鼓励这种积极的行为，以便在组织内推行良好的组织文化。

5、优美和善的语言：柔和的言谈创造和谐的环境。当然，说什么和怎么说是很有学问的，索尼认为每天从我们口中吐出的所有话语，都是非常重要的。只要所说的话是真实、和谐、有慈悲心和有意义的，它们从口里讲出来就具有价值。根据卡塔尼亚的说法 \[10\] ，行为可分为语言行为或非语言行为。这种认识与提升行为是有联系的，因为说话是一种语言行为，如果能以积极态度讲话，就会表现出积极的行为。最终，这一做法将促进个人发展，并提高HRD。

6、布施：慷慨布施是一种积极的行为，佛教也认为布施是一种好品行。从商业角度来看，慷慨大方的性格是必要的，因为员工这样做会促进积极行为的发展。由此，可以得出结论：布施有利于实现个人发展。

7、戒酒：醉酒和发狂会摧毁心智，因此人们应该保持清醒的头脑和健康的身体；如果员工在工作时喝醉酒，对公司来说是个大问题。一般情况下，醉酒之人无法像清醒的人那样作出积极的行为。根据健康和安全部门提示，任何员工都必须确保自己不受酒精的控制，因为在喝醉的情况下，酗酒者在工作中会危及自他的安全、健康或财产\[11\]。因此，戒酒可改善个人行为，最终达到组织层面的行为提升。

8、谦虚：这个因素强调不骄傲的重要性。根据索尼的说法\[9\]，假装的“谦虚”，只是骄傲的伪装。进一步来说，谦虚的人不自我宣传、身体或言语上不张扬、而是克制。在工作中对每个人都很谦虚属于行为的改善，因为根据卡塔尼亚的观点\[10\]，行为包括语言行为和非语言行为，而谦虚属于非语言行为。因此，这方面的行为提升会促进组织HRD中的个人发展。

9、知足：这种品性意味着，员工可以用平静、不抱怨的心态来接受环境与状况。有这种行为的人可以产生正能量。佛教教导人们要对自己拥有的衣服、食物、房屋和药品感到满足。维持和保持这个所谓的“满足”，并不会很难，但当你要求高于“满足”时，便会感到焦虑和担忧；因此每个人都必须自己决定，是想得到更多的快乐、还是更多的麻烦。在商业视角下，员工应该学着让自己满足，并最终使自己个人发展的行为得到改善；索尼也认为\[9\] ，知足是人的良好行为的一部分。

10、感恩：佛陀教导说:“世界上有两种人很难找到：一种人首先去做（一些善良或助人的事），另一种人懂得感激并认知（这种善意）。”没有感恩心的人会倾向于消极的行为，因此这一开示应落实到工作中去应用。索尼也认为，感恩能改善行动（行为），这属于个人发展的组成部分。

11、安忍：这是种重要的美德，事实上是最高美德之一，它包括宽容、原谅和忍耐。一个练习安忍的人将会产生积极的结果，因为健康和安全部门揭示\[11\]，在工作场所的安忍，可将意外事故降低为零和产生最少的冲突。安忍是人们的一种行为，当员工改善行为、行持安忍，就意味着个人发展中的行为得以改善。

12、接受建议：这是指“一个人可以很容易地交流、沟通或接受建议”，这意味着“这是一个愿意纠正错误的人”，同时也隐含了对批评的容忍、接受建议时的礼貌和感激。一个温顺的人利用这个机会学习，因此这种品行代表着积极的行为。从商业的角度来看，当员工善于听取别人意见来纠正自己错误，说明个人行为有了改进。

**四、管理方面的应用**

这部分内容揭示：如何在管理中应用从经中选定的各条开示，以获得个人发展，以及实现HRD的最终目标。因为研究者前面已经讨论了这些开示如何与个人发展的组成部分关联，这就阐明了从商业角度应该采用哪种类型的管理方法，以便在组织层面更好地应用这些开示，具体见表2。

| 表2. 管理方面的应用 |  |  |
| :--- | :--- | :--- |
| 序号 | 佛法概念 | 管理应用 |
| 1 | 不与愚者交往 | 创造学习环境，雇佣与组织文化相匹配的人。每个员工都有机会和和善的人打交道。 |
| 2 | 与智者交往 | 创建从上到下的交流渠道、传递知识。然后每个员工都会变得聪明，也能相互联系。 |
| 3 | 正确的指导思想 | 制定员工应遵循的道德规范，组织教导员工如何正确思考的培训。 |
| 4 | 尊敬那些值得尊敬的人 | 培养员工之间相互尊重的文化和道德品质，以营造良好的工作环境和关系。 |
| 5 | 优美和善的语言 | 让员工认识到友好话语的重要性，以创建和谐的企业文化。制定包含这一准则的道德规范。 |
| 6 | 布施 | 企业社会责任项目代表了企业的“布施”活动。管理机构可以得到企业员工对此项目的支持，最终令员工也参与到布施中来。 |
| 7 | 戒酒 | 在工作环境中禁止酒精和吸烟，将其列入强制性纪律。 |
| 8 | 谦虚 | 保持良好的员工管理关系，平等对待每个人，实行良好的申诉处理程序。 |
| 9 | 知足 | 通过道德培养计划，使员工能够满足于现有的资源。 |
| 10 | 感恩 | 在入职培训中，教导新员工认识在工作中感恩老员工的重要性。 |
| 11 | 安忍 | 组织培养谦虚态度的项目 |
| 12 | 接受建议 | 组织培养积极心态的培训，以便员工学会用正确的态度来处理所犯的错误。 |

**五、结论**

研究人员的目的是发现佛教与个人发展之间的关系，这是HRD的组成部分。HRD的主要目标是在组织中策划和设计活动，为其成员提供学习必要技能、知识和改善行为的机会，以满足当前和未来的工作需求。HRD包括职业发展、绩效管理、组织发展和个人发展。

佛教的《大吉祥经》宣说了人类在社会中的个人发展，而本研究则揭示了相关开示与HRD中个人发展之间的关系。

此外，研究人员揭示了选定的开示与个人发展的三个组成部分——技能、知识和行为改善之间如何建立联系；并指出每条开示在管理上如何落实应用，以说明如何在组织内部应用这些开示，以获得HRD中个人发展所需求的员工技能、知识和行为改善。

管理方面的应用可以总结为：创建学习环境、雇佣与组织文化相应的员工、建立清晰的沟通流程、制定道德规范、支持员工组织社会责任活动、在工作环境中禁止酗酒、建立健全申诉处理程序、适宜的入职培训方案、组织培养道德和积极态度培训。
